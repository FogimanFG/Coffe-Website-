# Coffee  — Website

Официальный сайт краснодарской кофейни, которой я делал на заказ. Создан для привлечения гостей и информирования о меню, акциях и локации.

## Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** — адаптивный дизайн, анимации
- **JavaScript** — интерактивные элементы (галерея, модальные окна)

## Ключевые функции

- Главная страница с анонсами акций
- Раздел «Меню» с фильтрацией по категориям
- Галерея фотографий интерьера и блюд
- Форма бронирования столиков
- Контактная информация и карта проезда
